장보기 리스트

  •  

    "구매할 목록을 한번에 관리"

     

     

    1. 구매를 원하는 식자재를 등록하고, 날짜 별 구매 리스트를 확인할 수 있습니다.

    2. 달력을 활용해 날짜 간 이동을 편리하게 할 수 있습니다.

    3. 식자재를 클릭하여 식자재들의 상세정보를 확인할 수 있습니다

    4. 삭제 버튼 또는 스와이프 딜리트를 통해 장보기 리스트를 간편하게 삭제할 수 있습니다.
